Prerequisites
Python 3.10+ (3.11 recommended)
Docker Desktop (recommended for full stack with Qdrant)
OpenAI API key
DocFinQA JSON files in data/raw/ (at least train; validation for evaluation)
Option A — Full project with Docker (recommended)
1) Setup environment
From project root:

cd "C:\Users\zm535\Desktop\Ziad Mohamed Gamal\Projects\FinLedger.AI"
Copy-Item .env.example .env
Open .env and set:

OPENAI_API_KEY=sk-your-real-key
VECTOR_BACKEND=qdrant
QDRANT_HOST=localhost
For Docker runtime, compose overrides QDRANT_HOST to qdrant automatically.

2) Put dataset files
Place files here:

data/raw/docfinqa_train.json
data/raw/docfinqa_validation.json (for evaluation)
3) Start all services
.\scripts\run_stack.ps1 -Detached
This starts:

Qdrant (6333)
FastAPI (8000)
Streamlit (8501)
4) Run ingestion (required once)
.\scripts\run_ingestion.ps1
This chunks data, creates embeddings, and upserts into Qdrant.

5) Use the app
UI: http://127.0.0.1:8501
API docs: http://127.0.0.1:8000/docs
Health check: http://127.0.0.1:8000/health
In Streamlit:

RAG Assistant tab → ask financial questions
Evaluation Dashboard tab → run Ragas metrics
Stop everything:

.\scripts\stop_stack.ps1
Option B — Run locally (without Docker)
1) Install dependencies
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
Set OPENAI_API_KEY in .env.

2) Start Qdrant locally
If using VECTOR_BACKEND=qdrant, Qdrant must be running on localhost:6333 (Docker or native install).

3) Ingest data
python -m src.pipeline.run_ingestion ingest --input-path data/raw/docfinqa_train.json
4) Start API (Terminal 1)
uvicorn api.main:app --host 0.0.0.0 --port 8000
5) Start UI (Terminal 2)
streamlit run ui/app.py --server.port 8501
Open http://127.0.0.1:8501.

Optional CLI commands (advanced)
Single RAG query
python -m src.retrieval.run_query query --question "What was the revenue growth and which table supports it?"
Run Ragas evaluation
python -m src.evaluation.run_evaluation evaluate --input-path data/raw/docfinqa_validation.json --sample-size 50 --output-path data/processed/evaluation/latest_report.json
Report output:

data/processed/evaluation/latest_report.json
Expected pipeline flow
DocFinQA JSON
   -> ingestion (chunk + embed)
   -> Qdrant/FAISS index
   -> hybrid retrieval
   -> RAG answer with citations
   -> Ragas evaluation metrics
Common issues
OPENAI_API_KEY is required

Key missing in .env (not .env.example)
Query returns weak answers

Ingestion not run yet, or wrong dataset path
Qdrant connection error

Qdrant not running, or wrong QDRANT_HOST / QDRANT_PORT
Streamlit cannot reach API

API not running on port 8000
For Docker UI, API_BASE_URL=http://api:8000 is set by compose
Minimum “full run” checklist
.env created with valid OPENAI_API_KEY
Dataset placed in data/raw/
Stack started (docker compose or API+UI locally)
Ingestion completed successfully
Open UI and test /query + /evaluate
